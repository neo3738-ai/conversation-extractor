# Conversation Extractor

Extract, search, browse, and analyze Claude conversation history from a local SQLite database.

## Features

- **GUI browser** — desktop app to browse, search, and read your Claude conversations
- **CLI tool** — command-line extraction and querying
- **Full-text search** — find conversations by content, date, or tool usage
- **Statistics** — token counts, tool usage breakdowns, conversation analytics
- **Local only** — all data stays on your machine, nothing sent anywhere

## Binaries

Ships two executables:
- `conversation_extractor_gui` — desktop GUI application
- `conversation_extractor` — command-line tool
